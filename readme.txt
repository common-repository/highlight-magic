Highlight Magic is a WordPress plugin that allows logged-in users to highlight text on your website. 

== Description ==

**Highlight Magic** is a WordPress plugin that allows logged-in users to highlight text on your website. Users can select text passages, choose highlight colors, and manage their highlights across sessions. This plugin enhances user engagement by enabling personalized interaction with your content.

**Features:**

- **Text Highlighting:** Users can highlight any text on posts or pages.
- **Color Selection:** Choose from multiple highlight colors to organize or categorize highlights.
- **Persistent Highlights:** Highlights are saved and persist across page refreshes and user sessions.
- **Edit and Delete Highlights:** Users can change the color of existing highlights or remove them entirely.
- **User-Specific Highlights:** Each user's highlights are private to them; users cannot see others' highlights.
- **Customizable Interface Alignment:** Administrators can set the alignment of the highlight controls (top, right, bottom, left).

== Installation ==

1. **Upload Plugin Files:**
   - Upload the `highlight magic` folder to the `/wp-content/plugins/` directory.
   - Alternatively, install the plugin through the WordPress plugins screen directly by searching for "HighLighter".

2. **Activate the Plugin:**
   - Activate the plugin through the 'Plugins' screen in WordPress.

3. **Configure Settings (Optional):**
   - Go to 'HighLight Magic' settings under the WordPress admin menu to adjust alignment settings if necessary.

== Frequently Asked Questions ==

= Does HighLight Magic work for non-logged-in users? =

No, HighLight Magic only allows logged-in users to create and manage highlights.

= Can users see each other's highlights? =

No, highlights are user-specific. Each user can only see and manage their own highlights.

= How do I change the alignment of the HighLight Magic interface? =

You can change the alignment by going to 'HighLight Magic' settings in the WordPress admin dashboard and selecting your preferred alignment.

= Does HighLight Magic work with custom post types? =

Yes, HighLighter works with any post or page that uses the standard WordPress content editor.

= Will highlights persist if I edit the post content? =

If you significantly change the content of a post, existing highlights may not align correctly, as they are based on text offsets. It's recommended to inform users if you plan to make substantial edits.

= Will highlights work on custom post =

At present , it does not work on custom posts . But that is something I will try to optimize for custom post in future.
